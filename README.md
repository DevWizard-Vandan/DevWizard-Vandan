### ⚡ [HFT-Stack](https://github.com/DevWizard-Vandan/HFT-Stack)

> *The Unified Infrastructure*

A complete high-frequency trading infrastructure built from scratch — combining a **12.8M match/sec** execution engine with a distributed vector database. Fully integrated, fully documented, production-architected.

**→ [Live Architecture Showcase](https://devwizard-vandan.github.io/HFT-Stack/)**

<td width="50%" valign="top">

### 🗡️ [Titan](https://github.com/DevWizard-Vandan/Titan)

> *The Matching Engine*

Low-latency, single-threaded, lock-free **Limit Order Book (LOB)** and matching engine in Rust. Designed for the sub-microsecond execution demands of real trading systems — no GC pauses, no contention, no mercy.

![Rust](https://img.shields.io/badge/Rust-000000?style=flat-square&logo=rust&logoColor=white)
![Lock-Free](https://img.shields.io/badge/Lock--Free-Architecture-f093fb?style=flat-square)

</td>

</tr>
<tr>

<td width="50%" valign="top">

### 💎 [Vajra](https://github.com/DevWizard-Vandan/Vajra)

> *The Vector Database*

Distributed, fault-tolerant **in-memory vector database** built from first principles in Rust. Implements Raft consensus for strong consistency, HNSW for approximate nearest-neighbor search, and WAL for durability.

![Rust](https://img.shields.io/badge/Rust-000000?style=flat-square&logo=rust&logoColor=white)
![Raft](https://img.shields.io/badge/Raft-Consensus-764ba2?style=flat-square)
![HNSW](https://img.shields.io/badge/HNSW-Search-764ba2?style=flat-square)

</td>

<td width="50%" valign="top">

### 🧠 [Ritam](https://github.com/DevWizard-Vandan/ritam)

> *The Prediction Engine*

Self-improving **multi-agent AI system** for real-time Nifty 50 prediction. Combines sentiment analysis, historical analog matching, and reinforcement learning in a feedback loop that evolves with every market session.

![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![RL](https://img.shields.io/badge/Reinforcement-Learning-667eea?style=flat-square)
